👉 Qu’est-ce qu’un crédit participant ?

🔹 Définition

Un crédit = un participant inscrit à l’un de vos événements.
Peu importe le nombre d’événements créés (illimités 🚀), seul le nombre de participants décomptés consomme vos crédits.

📌 Exemple concret

🎤 Vous organisez 3 événements :
- Concert A : 30 inscrits
- Conférence B : 50 inscrits
- Atelier C : 20 inscrits

Total = 100 participants → Vous consommez 100 crédits, même si vous avez créé 3 événements.

⚡ Avantage

Vos événements sont totalement illimités ✅. Vous payez uniquement pour les vrais participants, ce qui rend QR Event ultra flexible.

📲 Module complémentaire : ScanEvent

🎟️ Contrôle d’accès rapide

Avec ScanEvent, scannez facilement les QR codes de vos participants depuis un smartphone, une tablette ou un PC équipé d’une caméra.

⚡ Anti-fraude

Chaque QR code est unique et peut être validé une seule fois. Impossible de le partager ou de le falsifier ✅.

📊 Suivi en direct

À chaque scan, l’information est envoyée vers votre tableau de bord. Vous voyez en temps réel qui est déjà entré et combien de places restent.

🙌 Facile à déployer

Aucune installation complexe : ScanEvent est accessible directement depuis votre navigateur (Chrome, Safari, Edge).
Idéal pour vos équipes d’accueil.
